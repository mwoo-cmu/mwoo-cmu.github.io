  1. Split up the character into various disconnected parts.
  2. Each disconnected part is written using some radicals.
  3. These radicals are numbered from left to right, from top to bottom, from outer to inner.
  4. If there's only one part, just use those radicals.
  5. If there's more than one part, the first part uses its first and last radicals. (If there's only one radical, just use that.)
  6. If there's only two parts, the second part uses its first, second, and last radicals.
  7. If there's more than two parts, the second part uses its first and last radicals, then add the last part's last radical.
  8. If there are outer and inner parts, and the outer is not its own part, don't add the inner parts, just the outer '田tin4' or '月jyut6'.
